# North

Excel VBA マクロ集

## VBAマクロ一覧

### UnifySheetFormatting.bas
全シートのフォントと表示倍率を統一するマクロです。

**機能:**
- 全シートのフォントを統一（デフォルト: 游ゴシック、サイズ11）
- 全シートの表示倍率を統一（デフォルト: 100%）

**使用方法:**
1. Excel VBAエディタ（Alt+F11）を開く
2. ファイル → インポートから `UnifySheetFormatting.bas` をインポート
3. マクロを実行（F5キー または ツール → マクロ → マクロから選択して実行）

**カスタマイズ:**
コード内の以下の値を変更することで、設定を調整できます：
- `targetFont`: フォント名（例: "游ゴシック", "メイリオ", "MS ゴシック"）
- `targetSize`: フォントサイズ（例: 10, 11, 12）
- `targetZoom`: 表示倍率（例: 80, 100, 120）

### AddRowsAndGoHome.bas
全シートの先頭に2行追加し、最初のシートに移動するマクロです。

### UpdateSheetList.bas
全シートの一覧をハイパーリンク付きで自動生成するマクロです。
