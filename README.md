# WebXR オセロ (Othello VR)

WebXRに対応した3Dオセロゲームです。VRヘッドセットを使用してプレイすることも、通常のブラウザでプレイすることもできます。

## 機能

- **3D盤面表示**: Three.jsを使用した美しい3D表示
- **WebXR対応**: VRヘッドセットでの没入プレイ
- **CPU対戦**: 3段階の難易度設定（Easy/Normal/Hard）
- **効果音**: Web Audio APIによる効果音
- **アニメーション**: 駒を置く・ひっくり返すアニメーション
- **ヒント表示**: 有効な手をハイライト表示

## 技術スタック

- **フレームワーク**: Vite + TypeScript
- **3Dレンダリング**: Three.js
- **XR**: WebXR Device API
- **オーディオ**: Web Audio API

## インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# プロダクションビルド
npm run build
```

## 使い方

### 通常プレイ

1. ブラウザでゲームを開く
2. 難易度を選択（Easy/Normal/Hard）
3. 「ゲームスタート」をクリック
4. 緑色にハイライトされたセルに駒を置く

### VRモード

1. VRヘッドセットを接続
2. 「VRモードで開始」をクリック
3. コントローラーのトリガーで駒を置く

## 操作方法

### マウス/タッチ
- **クリック**: 駒を置く
- **ホバー**: 有効なセルをハイライト

### VRコントローラー
- **トリガー**: 駒を置く
- **ポインター**: 盤面を指す

## ゲームルール

オセロ（リバーシ）の標準ルールに従います：

1. 黒が先手
2. 相手の駒を挟むと、挟まれた駒が自分の色になる
3. 有効な手がない場合はパス
4. 盤面が埋まるか、両者とも打てなくなったらゲーム終了
5. 駒が多い方の勝ち

## AI難易度

- **Easy**: ランダムに近い手を選択
- **Normal**: 位置評価と貪欲法を使用
- **Hard**: ミニマックス法 + αβカットによる先読み
