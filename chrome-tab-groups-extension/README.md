# Tab Groups Cleaner

タブグループを一括削除・管理できるChrome拡張機能です。

## 機能

- **全グループを削除**: 全てのタブグループとその中のタブを一括削除
- **全グループを解除**: グループを解除してタブは保持（タブを閉じずにグループのみ解除）
- **個別グループ操作**: 各グループを個別に削除または解除
- **グループ一覧表示**: 現在のタブグループを色付きで一覧表示

## インストール方法

1. このフォルダをダウンロードまたはクローン
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `chrome-tab-groups-extension` フォルダを選択

## 使い方

1. Chromeツールバーの拡張機能アイコンをクリック
2. 現在のタブグループ一覧が表示される
3. 以下の操作が可能：
   - **全グループを削除**: 全てのグループと中のタブを削除
   - **全グループを解除（タブは保持）**: グループのみ解除し、タブは残す
   - **個別操作**: 各グループの「削除」「解除」ボタンで個別に操作

## 背景

「Acid Tabs」などのタブグループ管理拡張機能を使用していると、タブグループが無限に生成されてChromeが重くなる問題が発生することがあります。この拡張機能は、そのような状況を解決するためのシンプルなツールです。

## 技術仕様

- Manifest V3対応
- Chrome TabGroups API使用
- 権限: `tabGroups`, `tabs`
